> 🚀 使用 AI (SOLO) 在 20 分钟内从零搭建的零代码数据可视化工具

## ✨ 特性

- 📤 **拖拽上传** - 支持拖拽或点击上传 CSV 文件
- 📊 **4 种图表类型** - 柱状图、折线图、饼图、散点图
- ⚙️ **灵活配置** - 自定义图表标题、数据列选择
- 💾 **一键导出** - 导出 PNG 图片
- 🎨 **美观界面** - 现代科技感设计，流畅动画
- 📱 **响应式布局** - 适配桌面和移动设备
- 🔧 **零代码** - 全程通过自然语言对话完成开发

## 🚀 快速开始

### 安装

```bash
# 克隆仓库
git clone https://github.com/namune/dataviz-pro.git

# 进入目录
cd dataviz-pro

# 安装依赖
npm install

# 启动开发服务器
npm run dev
```

### 使用

1. 正确下载并应用node_modules 正确搭建
2. 点击"加载示例数据"或拖拽上传 CSV 文件
3. 选择图表类型和配置
4. 点击"导出"下载图表

## 💡 使用流程

```
上传数据 → 预览数据 → 选择图表类型 → 配置图表 → 导出图片
```

## 🛠️ 技术栈

- **前端框架**: React 18 + TypeScript
- **构建工具**: Vite
- **样式**: Tailwind CSS
- **图表库**: Chart.js + react-chartjs-2
- **CSV 解析**: Papa Parse
- **状态管理**: Zustand
- **图标**: Lucide React

## 🎯 项目背景

这是一个使用 AI (SOLO) 辅助开发的实战项目。在日常工作中，数据分析和可视化是常见需求，但每次都要写代码或用 Excel，效率很低。通过这个项目，我探索了如何利用 AI 工具快速构建实用的数据可视化工具。

**核心痛点解决**：
- ❌ 数据分析需要写代码，门槛高
- ❌ Excel 图表样式单一
- ❌ 没有简单易用的零代码工具
- ✅ **用 AI 在 20 分钟内解决以上所有问题**

## 🔥 功能演示

### 数据上传
支持拖拽或点击上传 CSV 文件，自动解析并预览数据。

### 图表生成
支持 4 种常用图表类型：
- 📈 柱状图 - 适合比较不同类别的数据
- 📉 折线图 - 适合展示趋势变化
- 🥧 饼图 - 适合展示占比关系
- ⚡ 散点图 - 适合展示相关性

### 灵活配置
- 自定义图表标题
- 选择 X 轴标签列
- 多选数据列
- 一键导出 PNG
